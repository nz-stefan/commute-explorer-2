Project

There and back again

This web application was submitted to the "There and Back Again" data visualisation competition organised by Stats NZ to find an original, fun, and interactive visual representation of the 2018 Census commuter data.

I wanted to create an awesome-looking, map based web application that captures the user's interest long enough to spent time playing and engaging with the data. The UX was very important to achieve this goal. Thus a significant chunk of development time was invested into tweaking colours, fonts and the arrangement of UI elements.

Plain numbers are often hard to comprehend and so I made some effort to make commuter figures more relatable. The mode of travel histograms give an impression of how fractions of commuters distribute across the various places of residence and work. I was surprised to see the low number of commuters using public transportation and that only very few places of work (mostly in the CBDs of the higly populated regions) seem to benefit from the availability of trains, buses and ferries. There are more people working at home than there are commuters using public transport. Sadly, most of our commutes still rely mostly on cars.

Data

The data behind this application is available in the Datafinder hosted by Stats NZ. Additionally, I used the Statistal Area 2 2018 data set which contains shape files of the statistical areas referenced in the commuter data set. I simplified the polygons in those shape files to reduce the amount of data to display in the browser for this application. I also processed the Geographic Areas File 2020 to group statistical areas into the main NZ regions.

Application

This application is developed using the R Shiny web application framework. I like beautiful data products and so I used HTML templates to create a unique look for this application. The template is called Titan and is freely available for download. There was a bit more JavaScript foo necessary to customise some of the visualisations.

Deployment

The development environment of this project is a Rocker based Docker container running the Open Source version of RStudio Server. Docker allows to control all aspects of the development environment, i.e. operating system, system packages and R version. The versions of R packages are tracked using the renv package. The containerisation intends to simplify the setup of a working development environment.

The application itself is hosted on RStudio's shinyapps.io which offers push button deployment and a generous free tier.

All code developed for this application is open source and available on GitHub.

Themes & Assets

The amazing theme used in this application is called Titan and is developed by ThemeWagon. It can be downloaded for free!

The hero image on the top of this application was created by Harely Davidson and made available at Unsplash for free.

Charts & Graphs

The beautiful ECHARTS engine is behind all charts used in this project. The engine is very comprehensive, supports many chart types and comes with a selection of stylish themes. John Coene developed an easy to use R wrapper complete with Shiny support, documentation and impressive examples.
